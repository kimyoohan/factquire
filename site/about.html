What This Is

ModelWire is a machine-readable facts feed for commercial LLM APIs. It tracks model identifiers, pricing, lifecycle status, token limits, modalities, and source evidence.

Sourcing Policy

Every entry uses primary sources only: official provider docs, pricing pages, changelogs, or official blogs. Each source includes a verbatim quote and the fields that quote supports. Unconfirmed fields remain null and are listed in gaps.md.

Corrections

Corrections should be made as source-backed changes to data/facts.json, then validated and rebuilt. The feed is regenerated from the checked data file so downstream consumers can diff changes.

Programmatic Use

Fetch /feed.json. The response wrapper contains generated_at, count, version, and models. Each item in models validates against schema/model_fact.schema.json.
